## Objetivo

O projeto **Mente Viva** consiste no desenvolvimento de uma plataforma digital com jogos interativos voltados para **idosos residentes em asilos**, com o objetivo de promover:

- Estímulo cognitivo  
- Entretenimento  
- Bem-estar emocional  
- Socialização  

A plataforma foi projetada para ser **intuitiva, acessível e compatível com limitações motoras e cognitivas**.

---

## Justificativa do PAC perante a entidade beneficiada

O **Lar dos Idosos de São Francisco do Sul** necessita de atividades que apoiem o desenvolvimento cognitivo e emocional dos residentes. Muitos enfrentam desafios como:

- Dificuldades de memória  
- Pouca interação social  
- Falta de atividades recreativas adequadas  

O **Mente Viva** oferece uma solução digital que:

- Estimula memória, atenção e raciocínio  
- Promove bem-estar e diversão  
- Incentiva autonomia através de interfaces simplificadas  
- Auxilia cuidadores com uma ferramenta moderna de apoio terapêutico  

O projeto gera impacto social efetivo, alinhado aos objetivos do PAC.

---

## Descrição do App

A plataforma possui **cinco jogos principais**, desenvolvidos com base em práticas de estimulação cognitiva para idosos.

### 1. Jogo da Memória (Pareamento de Cartas)
- Encontre pares iguais.  
- **Benefícios:** memória, foco e paciência.  
- **Dificuldade:** ajustável.  

### 2. Caça-Palavras Online
- Encontre palavras ocultas.  
- **Benefícios:** vocabulário, atenção e memória.  
- **Temas:** músicas, culinária, tradições.

### 3. Pintura/Colorir Digital
- Preencha desenhos com cores.  
- **Benefícios:** coordenação motora, criatividade e relaxamento.  
- Interface com botões grandes e cores vivas.

### 4. Quiz de Perguntas e Respostas
- Perguntas de múltipla escolha.  
- **Temas:** cultura dos anos 50–70, novelas, culinária.  
- **Benefícios:** memória de longo prazo, autoestima.

### 5. Quebra-Cabeça Digital (Puzzle)
- Monte imagens arrastando peças.  
- **Benefícios:** percepção visual e raciocínio espacial.

## Requisitos para preparar o ambiente de desenvolvimento

### Tecnologias utilizadas
- Node.js 18+  
- NPM  
- Framework baseado em React/Vite  

### Passos de instalação

1. **Clonar o repositório**
```bash
git clone https://github.com/Uma-Hora-da-Certo/Mente-Viva.git
```
Entrar na pasta
```bash
cd mente-viva
```

Instalar dependências
```bash

npm install
```

Rodar o servidor
```bash
npm run dev
```

Acessar o projeto no navegador
```bash
http://localhost:5173/
```
